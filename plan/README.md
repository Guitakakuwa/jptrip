# Japan Trip — Sep 12–27, 2026

3 American guests + Gui (host). Haneda arrival Sep 12 @ 15:20. Haneda departure Sep 27 @ 10:45.

---

## How to use these docs

This is a **menu**, not a schedule. Pick items based on energy each day.

Every experience is rated:
- **Effort 1–5** (1 = sit and look, 5 = long active day)
- **Exhaustion** Low / Med / High
- **From base** (min from Shibuya by train)
- **Time** (hours, including transit)
- **Reserve?** Yes / No / Recommended

Food has **3+ options per meal per area**. Pick what you crave.
